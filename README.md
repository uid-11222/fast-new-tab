# fast-new-tab #

[![NPM](https://nodei.co/npm/fast-new-tab.png)](https://nodei.co/npm/fast-new-tab/)

**fast-new-tab** get npm download stats for chosen packages and save it to JSON files.

## Usage ##
You need a node version >=6.0.0.
